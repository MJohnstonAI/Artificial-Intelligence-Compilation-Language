# Trademark and Project Identity Policy

The names **AICL** and **Artificial Intelligence Compilation Language**, together with the canonical project identity and official project marks, are used to identify the original public research project originated by Marc Johnston.

## What is allowed
You may:
- refer to this project by name
- state that your implementation is "compatible with AICL"
- build tools, apps, or derivative implementations based on AICL

## What is not allowed
You may not:
- falsely claim to be the originator of AICL
- imply your fork or derivative is the canonical official AICL project if it is not
- use project identity in a misleading way

## Note
This policy is intended to preserve clarity of origin and public understanding. It is not intended to block legitimate open-source use or derivative implementation work.

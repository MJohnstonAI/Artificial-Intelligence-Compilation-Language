# AICL Consortium Working Draft

This file is the collaborative design layer for AICL.

It exists between the public landing documents and the more formal specification artifacts. Use it for convergence notes, candidate wording, and draft-level synthesis that is not yet ready to be treated as the strongest technical baseline.

## Intended Use

Good content for this file:

- draft synthesis notes
- unresolved integration questions
- candidate wording before promotion into the manual or unified spec
- commentary-derived issues that still need textual reconciliation

Avoid using this file as a dumping ground for unrelated repo notes or vague marketing copy.

# Contradiction Report Template

Use this template when reporting a focused contradiction or internal inconsistency.

## Summary

Briefly state the contradiction.

## Files Involved

List files and sections involved.

## Conflicting Claims

Quote or summarize the claims that conflict. Keep quotations short and precise.

## Why This Matters

Explain the effect on semantic authority, proof obligations, runtime behavior, tooling, schemas, examples, governance, or public positioning.

## Proposed Minimal Correction

State the smallest correction that resolves the contradiction.

## Does This Preserve AI-Native Intent?

Explain whether the correction preserves AICL's design objective: AI-native semantic compilation for AI systems.

## Does This Accidentally Move AICL Toward Legacy-Language Assumptions?

Explain whether the correction makes AICL more like a conventional human-first language, prompt system, or DSL in a way that weakens the project.

# Public Claim Guide

## Safe claims

AICL is an open research project exploring AI-native semantic programming for agentic software construction.

AICL treats software construction as semantic compilation from intent, constraints, capabilities, policies, workflows, and proof obligations toward materialized artifacts.

AICL is research-stage and should be evaluated as a language-design and compiler-architecture proposal, not as a production compiler.

## Claims to avoid

Do not say:

- AICL replaces all programming languages.
- AICL is production ready.
- AICL guarantees safe autonomous agents.
- AICL is officially integrated with OpenClaw.
- AICL has a certified proof checker.
- AICL can compile every app today.

## Preferred short description

AICL is an open research project for an AI-native semantic programming language focused on intent, constraints, capabilities, policies, proofs, workflows, and materialized artifacts.

## Longer project description

AICL is an open research project exploring how AI systems might specify software through semantic contracts rather than syntax-first programming alone. It focuses on intent, constraints, policies, capabilities, proof obligations, workflows, WKG-backed grounding, and materializers that can eventually produce conventional deployable artifacts.

## For academic audiences

Present AICL as a research-stage programming-language and compiler-architecture project. Invite critique of the type system, proof boundaries, WKG grounding, operational semantics, schemas, examples, and relationship to prior work.

## For agentic AI audiences

Present AICL as a proposed semantic substrate for bounded agent workflows, capability-aware execution, proof-aware planning, and materialized outputs. Avoid implying guaranteed agent safety.

## For programming language audiences

Present AICL as an AI-native semantic language proposal whose current value is in its specification, examples, schemas, and open problems. Avoid framing it as a replacement for established implementation languages.

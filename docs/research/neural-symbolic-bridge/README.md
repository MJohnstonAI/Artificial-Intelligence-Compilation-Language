# Neural-Symbolic Bridge Research Track

## Purpose

This track explores whether AICL's symbolic SHG can be projected into differentiable or vector-backed representations for external model analysis.

The goal is to investigate neural-symbolic interoperability without weakening the canonical kernel.

## Boundary

The symbolic SHG remains the source of truth.

Any differentiable or vector projection is experimental, derived, and externally validated. It must not become an independent semantic authority over ICC, WKG grounding, proof obligations, HAIG decisions, PACT coordination, or materializer behavior.
